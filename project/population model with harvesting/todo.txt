## BSc Project: Logistic Population Models with Harvesting

### 1. Mathematical Analysis (Theory Section)

| Item | What to Show |
|------|-------------|
| Logistic ODE derivation | Start from Malthus model → add carrying capacity → arrive at logistic equation |
| Constant harvesting model | Derive `dP/dt = kP(1 - P/N) - a` and explain each parameter |
| Periodic harvesting model | Derive `dP/dt = kP(1 - P/N) - a(1 + sin(bt))` and explain a, b |
| Equilibrium derivation | Show full quadratic derivation → get P1, P2 formulas by hand |
| Stability analysis | Show P1 is stable, P2 is unstable using `dF/dP` sign test |
| MSY derivation | Derive Maximum Sustainable Yield = `kN/4` step by step |
| Discriminant condition | Show when `disc > 0`, `= 0`, `< 0` and what each means biologically |

### 4. Analysis & Discussion Points

| Topic | What to Discuss |
|-------|----------------|
| Biological interpretation of P1 | Safe equilibrium fish population the fishery settles to |
| Biological interpretation of P2 | Danger threshold — if population falls below this, extinction follows |
| MSY significance | Maximum harvest a fishery can sustain indefinitely |
| Over-harvesting consequence | What happens when a > MSY — link to real-world fishery collapse |
| Periodic vs constant comparison | Which model is more realistic, which allows more recovery time |
| Effect of initial condition | Does the starting population matter? When does it not? |
| Sensitivity analysis | Which parameter (k, N, a, b) affects the system most? |
